Nombre del proyecto
Hola Mundo en C

Descripción:
Este proyecto consiste en un programa básico en lenguaje C que muestra el mensaje "Hola mundo" en la pantalla. Su objetivo es practicar la compilación y ejecución de programas simples.

Prerrequisitos (tecnologías a instalar):
Compilador de C (GCC)
Visual Studio Code u otro editor de código


Pasos para su ejecución:
Descargar o clonar el repositorio.
Abrir la carpeta del proyecto en un editor de código.
Compilar el archivo del programa usando un compilador de C.
Ejecutar el programa desde la terminal.
El programa mostrará el mensaje "Hola mundo" en pantalla.
